tmux commands

tmux -> start tmux

ctrl b + d -> exit tmux

ctrl b,c -> new window

ctrl + b [0...9] -> switch to N window.

ctrl + b, [ -> scroll mode, pgUp pgDwn for scrolling 
  q to exit

## other notes:

If you ever modify .tmux.conf make sure you run this command:
  - `tmux source-file ~/.tmux.conf`
  - restarting your pc also works :)
